# 🛠️ Smart Resume AI - Tech Stack & Tools

A comprehensive overview of the technologies, frameworks, and libraries that power the **Smart Resume AI** ecosystem.

---

## 🎨 Frontend & User Interface
| Tool | Purpose | Key Features |
| :--- | :--- | :--- |
| **Streamlit** | Core Framework | Rapid deployment of interactive data applications in Python. |
| **Vanilla CSS3** | Custom Styling | Glassmorphims, premium dark mode, and topographic animations. |
| **Plotly** | Data Visualization | Interactive ATS score gauges and skill distribution charts. |
| **Lottie-Web** | Micro-animations | Lightweight JSON-based animations for engaging UX. |
| **Streamlit Extras** | UI Components | Option menus, custom navigation, and floating notifications. |

## 🧠 Artificial Intelligence & NLP
| Tool | Role | Description |
| :--- | :--- | :--- |
| **Google Gemini Pro** | Primary LLM | Powers deep resume analysis and personalized recommendations. |
| **Groq (Llama 3)** | Fallback LLM | Ultra-fast inference engine for redundancy and speed. |
| **spaCy** | NLP Analysis | Named Entity Recognition (NER) for skill/experience extraction. |
| **NLTK** | Text Processing | Used for tokenization, keyword density, and sentiment analysis. |
| **Scikit-Learn** | ML Utilities | TF-IDF vectorization and cosine similarity for job matching. |

## 📄 Document Processing & OCR
| Tool | Format | Capability |
| :--- | :--- | :--- |
| **python-docx** | .docx | Reading, parsing, and generating Word-based resumes. |
| **pdfplumber** | .pdf | High-precision text extraction from vector-based PDFs. |
| **pytesseract** | OCR | Optical Character Recognition for scanned/image-based resumes. |
| **pdf2image** | Conversion | Converting PDF pages into high-res images for OCR processing. |
| **Poppler** | Rendering | Essential system dependency for PDF-to-image conversion. |

## 🏗️ Backend & Infrastructure
| Tool | Layer | Implementation |
| :--- | :--- | :--- |
| **Python 3.11+** | Language | High-level logic and data orchestration. |
| **PostgreSQL** | Production DB | Containerized relational database for persistent storage. |
| **SQLite** | Local DB | Zero-config database for offline development. |
| **SQLAlchemy** | ORM | Database abstraction for seamless switching between SQL types. |
| **Docker & Compose** | Containerization | Standardized environment for deployment (Web + DB services). |

## 🛠️ Developer Tools & Utilities
- **Selenium**: Automated web interaction for job search aggregation.
- **dotenv**: Secure environment variable management for API keys.
- **Pandas**: Structural data manipulation for analytics and dashboarding.
- **Requests**: Handling communication with external AI APIs.
- **Pillow (PIL)**: Advanced image manipulation for resume thumbnails and OCR.

---
*This tech stack is optimized for performance, scalability, and modern AI integration.*
